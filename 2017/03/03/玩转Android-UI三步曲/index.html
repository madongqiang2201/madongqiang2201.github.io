玩转Android UI三部曲

本文是对将来一段时间要整理的android知识的索引，算是给自己挖个坑。

这系列文章的主题是玩转Android UI，跟UI直接相关的有三个主题分别是Drawable，动画和自定义控件。系统自带控件的使用方式不在本系列文章的讨论范围之内。所以被系列文章分为三个部分介绍：

  1. Android Drawable体系详解
  2. Android 动画体系详解
  3. Android自定义控件详解

Android Drawable体系详解

这部分内容主要包括对Android原有的各种Drawable资源的详细探讨，以及对Android5.X中新增的Drawable资源的详细探讨。

一、传统Drawable

  1. ColorDrawable
  2. BitmapDrawable
  3. NinePatchDrawable
  4. ScaleDrawable
  5. RotateDrawable
  6. GradientDrawable－－xml中<shape>标签对应的drawable
  7. StateListDrawable
  8. InsetDrawable
  9. ClipDrawable
  10. LayerDrawable
  11. LevelListDrawable
  12. TransitionDrawable
  13. AnimationDrawable
  14. 自定义drawable

二、android 5.X新增Drawable

  1. RippleDrawable
  2. AnimatedStateListDrawable 对应XML中<animated-selector>标签
  3. AnimatedVectorDrawable／VectorDrawable

Android动画体系详解

这部分包括的内容比较多，主要分为一下几个方面：

  1. 传统View动画解析
  2. Property属性动画解析
  3. 插值器Interpolator以及估计器TypeEvaluator解析
  4. LayoutAnimation动画解析
  5. Android 5.X新增动画解析

下面看一下每一个小部分包涵的详细内容：

一、View Animation

View Animation包括Tween(补间)动画和Frame(帧)动画

1、Tween Animation：

Tween动画有以下几个子类，我们都会详细介绍

  • AlphaAnimation
  • TranslateAnimation
  • RotateAnimation
  • ScaleTranslation
  • AnimationSet
2、Frame Animation

Frame动画实质是一种Drawable，并没有对View做任何操作，而是操作的Drawable

3、自定义动画

这部分内容我们将介绍如何自定义Android View动画

二、插值器Interpolator和估值器TypeEvaluator

这部分内容我们将讨论：

  1. 系统插值器和估值器的解释与使用
  2. 如何自定义插值器和估值器

三、属性动画

这部分内容主要包括：

  1. ObjectAnimator解析
  2. ValueAnimator解析
  3. AnimatorSet解析
  4. PropertiesValuesHolder与Keyframe解析
  5. ViewPropertyAnimator概述

四、LayoutAnimation动画

这部分内容解析针对布局的动画，这个动画作用在ViewGroup及其子类身上，动画效果会在你往ViewGroup中添加／删除子元素的时候显示在子元素身上。一共有三种方式应用LayoutAnimation：

  1. android:animateLayoutChanges=“true”使用默认Layout动画
  2. LayoutAnimationController／android:layoutAnimation使用自定义Layout Animation（View动画）
  3. LayoutTransition使用自定义ViewGroup Animator（属性动画）

四、Activity转场动画

  1. 传统转场动画详解

  2. Android 5.X Material转场动画：

    • 使用系统动画：Fade，Slide，Explode，有共享元素时可以指定共享元素过渡动画changeBound，changeClipBounds，changeImageTransform，ChangeTransform，ChangeScroll等
    • 自定义Transition

五、android 5.X动画

这部分介绍Android 5.X引入的动画效果：

  1. Reveal Effect，揭露效果
  2. 曲线动画Path／PathInterpolator
  3. StateListAnimator
